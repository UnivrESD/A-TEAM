Installation
============

Requirements
------------

  [1] CMake (2.8 or above)

  [2] C++ compiler and linker
  
  
Building and installing
-----------------------

  [1] Create a directory, and go into it:

  > mkdir build
  > cd build

  [2] Run CMake, with the path to this directory:

  > cmake ..

  [3] Compile using the following command:

  > make
